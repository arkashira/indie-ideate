# breakeven.md

## Cost per Active User
- **Compute**: $0.05/user/month (based on estimated cloud compute costs for running AI models)
- **Storage**: $0.02/user/month (based on storage costs for user data and generated ideas)
- **Bandwidth**: $0.03/user/month (based on data transfer costs for user interactions)

**Total Cost per Active User**:  
**$0.05 + $0.02 + $0.03 = $0.10/user/month**

## Customer Acquisition Cost (CAC) Range
- **Estimated CAC**: $30 - $50 per user (includes marketing, sales, and onboarding costs)

---

## Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**:  
  - Starter: $10/month  
  - Pro: $25/month  
  - Enterprise: $50/month  
- **Average User Lifespan**: 24 months  
- **LTV Calculation**:  
  - Starter: $10 * 24 = $240  
  - Pro: $25 * 24 = $600  
  - Enterprise: $50 * 24 = $1200  

**Weighted Average LTV** (assuming 50% Starter, 30% Pro, 20% Enterprise):  
LTV = (0.5 * $240) + (0.3 * $600) + (0.2 * $1200)  
LTV = $120 + $180 + $240 = **$540**

---

## Break-even Users Count
**Break-even Point**:  
- Fixed Costs (monthly): $1,000 (estimated operational costs)  
- Contribution Margin per User:  
  - Starter: $10 - $0.10 = $9.90  
  - Pro: $25 - $0.10 = $24.90  
  - Enterprise: $50 - $0.10 = $49.90  

**Break-even Users Count**:  
- Starter: $1,000 / $9.90 ≈ 101 users  
- Pro: $1,000 / $24.90 ≈ 40 users  
- Enterprise: $1,000 / $49.90 ≈ 20 users  

---

## Path to $10K MRR
To achieve $10,000 MRR, we can consider different combinations of tiers:

1. **Starter Tier**:  
   - Price: $10  
   - Users Needed: $10,000 / $10 = **1,000 users**

2. **Pro Tier**:  
   - Price: $25  
   - Users Needed: $10,000 / $25 = **400 users**

3. **Enterprise Tier**:  
   - Price: $50  
   - Users Needed: $10,000 / $50 = **200 users**

**Optimal Path**:  
- **Combination**: 300 Starter users + 200 Pro users + 100 Enterprise users  
- **Revenue**: (300 * $10) + (200 * $25) + (100 * $50) = $3,000 + $5,000 + $5,000 = **$13,000 MRR**

This combination not only achieves the $10K MRR target but also provides a balanced user base across different tiers.
