# Indie-Ideate
A guided onboarding wizard for generating ideas.

## Usage
1. Run `python -m pytest` to test the implementation.
2. Run `python src/onboarding.py` to use the onboarding wizard.

## Features
* Guided onboarding wizard with 3 steps: profile setup, interest input, and first idea generation.
* Progress bar indicates current step.
* User can skip or edit inputs before final submission.
* On completion, user lands on the Idea List page with results.
* Onboarding state is saved in local storage for returning users.
