# Indie-Ideate

Indie-Ideate is a step-by-step onboarding wizard for generating and validating ideas.

## Usage

1. Start the onboarding process by creating an instance of the `OnboardingWizard` class.
2. Input skills using the `input_skills` method.
3. Generate ideas using the `generate_ideas` method.
4. Validate ideas using the `validate_ideas` method.
5. Select a template using the `select_template` method.
6. Get the user profile using the `get_user_profile` method.
7. Skip the onboarding process using the `skip_onboarding` method.

## Testing

Run the tests using `pytest` command.
